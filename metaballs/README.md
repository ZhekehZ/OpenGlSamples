### Fractal example

* 3.3 core profile
* prereqs - conan, cmake
* deps - glfw, glew, imgui, glm

#### How to build and run:
1. run `build.(bat|sh)` 
2. `cd build`
3. run `[Release/]fractal[.exe]`
